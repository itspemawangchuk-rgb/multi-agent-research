# Multi-Agent Research Team

## Overview
This project is part of the GenZ AI Challenge. It demonstrates a multi-agent research system where different AI agents collaborate to solve problems efficiently.

## Objective
To build a scalable multi-agent setup where agents can research, analyze, and combine results into a final output.

## Agents Used
- Research Agent
- Analysis Agent
- Summary Agent

## Technologies
- Python
- GitHub
- AI agent framework
